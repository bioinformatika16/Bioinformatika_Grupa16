How to start this program in Linux:
1. Open terminal
2. Navigate to the program's folder
3. Type into terminal the following command: "mono Program.exe test.txt"

The terminal should display the results of the input from the text file named "test.txt".

If one would like to change the input text file, just change the argument into a path where the wanted file is stored.

***For any issues running "mono" command in Linux, please refer to the Linux package website and download approriate package for Mono platform.
